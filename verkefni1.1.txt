1. Hvað gera eftirfarandi Linux skipanir?
cd - Stendur fyrir "change directory". Færir þig um möppu. t.d. cd Verkefni1/verkefni1.1
ls - Stendur fyrir "list". Skrifar niður allar möppur og öll skjöl í möppunni sem þú ert staddur.
pwd - Stendur fyrir "print working directory". Prentar út möppuna sem þú ert staddur í.
mkdir - Stendur fyrir "make directory". Býr til nýja möppu(a.k.a directory).
2. Hvað gera eftirfarandi Git skipanir?
git clone - Downloadar verkefni og allri útgáfu sögu verkefnisinns
git status - Skrifar allar nýjar eða breyttar möppur sem á eftir að committa
git diff - Sýnir munin á möppum
3. Hvað gera eftirfarandi Git skipanir saman? Hvaða gagn er að þeirri aðgerð?
git log - Sýnir lista af öllum commits sem þú hefur gert
git checkout - Skiptir yfir í tilgreinda grein og uppfærir directory-ið
git branch - Skrifar lista af öllum greinum
